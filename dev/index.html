MyTangent.numderiv — Function
numderiv(f, x₀, ε=1e-7)

Returns the appproximate derivative of ``f`` at ``x₀`` using the forward difference with step size ``ε``, namely

`` f'(x_0) \approx \frac{f(x_0 + \varepsilon) - f(x_0)}{\varepsilon} ``
